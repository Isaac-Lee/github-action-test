# GitHub Actions Markdown Pages 예제

이 문서는 `content/index.md`에서 작성되고, 빌드 시 HTML로 변환되어 배포됩니다.

## 포함 내용

- Markdown 기반 정적 페이지 생성
- GitHub Actions를 통한 빌드와 테스트
- GitHub Pages 배포 자동화

## 사용 순서

1. 저장소를 GitHub에 푸시합니다.
2. 저장소의 Pages 설정에서 **Build and deployment → Source**를 **GitHub Actions**로 둡니다.
3. `main` 브랜치에 푸시하면 자동으로 배포됩니다.
